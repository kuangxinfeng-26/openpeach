# Owner User Profile Template

## Identity

This file is the template for the primary owner profile in an OpenPeach family workspace.

Live runtime copies should live under:

```text
~/.openpeach/families/main/users/owner/user.md
```

## Stable Preferences

- OpenPeach should run on Linux.
- Windows is only for editing and version updates on the current development machine.
- Deployment should use npm.
- NLP models may be external.
- ASR and TTS may be local sidecars.
- Personal Telegram and WeChat are the first human channels.
- The system should stay lightweight enough for simple hardware.

## Privacy Rules

- Do not expose owner-private memory to family-shared memory without explicit permission.
- Do not treat one temporary preference as a permanent rule without evidence.
- Store sensitive facts as restricted memory candidates first.

## Runtime Notes

- This repository template should not contain personal secrets, phone numbers, bot tokens, API keys, or private device credentials.
